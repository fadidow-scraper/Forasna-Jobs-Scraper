# Forasna Jobs Scraper 💼🔍

### 📝 Project Overview
This project is a powerful **Python Web Scraper** specifically built to automate the collection of job listings from **Forasna**. The tool focuses on extracting comprehensive job data, providing users with a structured dataset that includes everything from job titles to detailed company requirements.

### 🚀 Key Features
*   **Detailed Job Extraction:** Captures Job Titles, Company Names, Locations, and Posting Dates.
*   **Experience & Requirements:** Specifically scrapes years of experience required and educational levels.
*   **Job Category Filtering:** Capable of fetching jobs based on specific industries or categories.
*   **Structured Data Output:** Automatically exports all scraped data into clean **Excel** or **CSV** formats for easy filtering and analysis.
*   **Efficiency:** Designed to handle pagination and scrape multiple pages of job results seamlessly.

### 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Libraries:** 
    *   `BeautifulSoup4`: For robust HTML parsing and data extraction.
    *   `Requests`: For managing HTTP networking and fetching page content.
    *   `Pandas`: For advanced data structuring, cleaning, and exporting to Excel/CSV.

### 📖 How to Run
1. **Clone the repository:** `git clone [Your-Repo-Link]`
2. **Install dependencies:** `pip install requests beautifulsoup4 pandas`
3. **Run the script:** Execute `python forasna_scraper.py`
4. **Access Data:** Check your project folder for the generated `.csv` or `.xlsx` file containing the job leads.
